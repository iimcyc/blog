# 编码

* encodeURI(), decodeURI()

  转义除了:

  | 类型         | 包含                        |
  | ----------- | --------------------------  |
  | 保留字符     | ; , / ? : @ & = + $         |
  | 非转义的字符  | 字母 数字 - _ . ! ~ * ' ( )  |
  | 数字符号     | #                           |

* encodeURIComponent(), decodeURIComponent()

  转义除了:

  **字母 数字 ( ) . ! ~ * ' - _**
