# 数据类型

最新的 ECMAScript 标准定义了 8 种数据类型:

* 7 种原始类型:
  * Null
  * Undefined
  * Boolean
  * String
  * Number
  * Bigint
  * Symbol
* 1 种引用类型:
  * Object
    * "标准的"对象  
      key(string|symbol): value(any)
    * 函数  
      函数是一个附带可被调用功能的常规对象。
    * 日期
    * 有序集: 数组和类型数组
    * 键控集: Maps, Sets, WeakMaps, WeakSets
    * 结构化数据: JSON
    * 标准库中更多的对象
